## Hello World!

##### C
```c
#include<stdio.h>

main()
{
	printf("Hello World\n");

}
```

##### C++
```cpp
#include <iostream>

int main() 
{
    std::cout << "Hello, World!\n";
    return 0;
}
```

##### C#
```c#
using System;

class Program
{
	static void Main(string[] args)
	{
		Console.WriteLine("Hello World!");
	}
}
```

##### Java
```java
public class Main {
	public static void main(String[] args) {
		System.out.println("Hello World!");
	}
}
```

##### Javascript
```javascript
alert("Hello World!");
console.log("Hello World!");
document.write("Hello World!");
```

##### Pascal

##### PHP
```php
echo "Hello World!\n";
```

##### Python
```python

```

##### Ruby
```ruby
puts("Hello World!")
```
